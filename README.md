Using Microsoft Foundry to create a specialized AI Agent:


Download the files from the two folders in the repository. 

Open the folders in Visual Studio Code

Go to the mslearn-agent-quickstart folder, computer-history-client folder, then click app.py

Open command palette in visual studio code, "Python: Create Terminal" to make a new python terminal

In the python terminal, type "cd computer-history-client"

then type python app.py to run the computing history agent.

ask the computing history agent questions about only computing information, any other type of questions will not be answered.
